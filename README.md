# Generate Configuration Scripts MCv2 - CPEs

## Description

This project is intended to generate the CPE configuration files.

## Pre-execution

With your favorite text editor, modify variables `PONTOS` and `PCGs` in file `gerarScripts.sh`.

Example:

```bash
[...]

PONTOS=2
PCGs=30

[...]
```
